# Custom Default Project

02Engine can replace the default blank project with one you choose. This is useful for classrooms, template projects, engine starters, extension packs, or personal workspaces.

## Set a Default Project

1. Open or create the project you want to use as the template.
2. Open **Settings**.
3. Choose **Set as Default Project**.

New editor sessions will start from that saved project instead of the built-in blank project.

## Restore the Original Default

Open **Settings** and choose **Restore To Original Default Project**.

## What To Put In a Template

Good default projects often include:

- Preferred stage size or runtime settings.
- Common sprites, costumes, sounds, and variables.
- Starter custom blocks.
- Trusted custom extensions.
- Comments explaining how to use the template.

## Compatibility

The default project is an editor preference. It does not change existing projects and does not affect exported `.sb3` files unless you create a new project from the template and save it.
